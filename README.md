  <p>一个基于纯 HTML/JS/CSS 开发的轻量级双屏提词器项目，开箱即用，无需后端。</p>

## ✨ 核心特性

本项目采用了现代化的苹果风扁平化设计，遵循“方便操作、直观、美观”的原则，提供低噪声、清晰分组和快捷操作的极致 UI 体验。

- **🖥️ 双屏联动同步**：项目分为“控制面板（导播屏）”与“显示窗口（演讲者屏）”，支持跨浏览器窗口的实时通信与状态同步。
- **📏 尺寸自适应预览**：副屏（显示窗口）会实时上报其实际显示区域的尺寸，并在主屏（控制面板）中实现等比例的精准预览。默认方向为不旋转（orientation=none）。
- **▶️ 多种播放模式**：
  - 单次播放结束（Once）：默认模式，播放完当前脚本即停止。
  - 单条循环（SingleLoop）：当前脚本播完后循环播放。
  - 顺序播放（Sequence）：播完当前脚本后自动播放下一条脚本。
- **📝 完善的脚本管理**：支持脚本的创建、编辑和删除。列表清晰展示脚本名称、一句话预览与创建时间。
- **⚡ 开箱即用**：纯前端实现（无任何第三方依赖库、无服务端），支持主流现代浏览器。

## 🚀 快速开始

### 1. 获取代码
将本项目克隆或下载到本地计算机：
```bash
git clone https://github.com/MUSSOKINA/shuangpingticiqi.git
```

### 2. 启动控制面板
使用现代浏览器（推荐使用 Chrome 或 Edge）双击打开 `双屏提词器-控制面板.html` 文件。

### 3. 开启提词器副屏
在控制面板中，点击打开/连接显示窗口，浏览器会弹出一个新的窗口，即 `双屏提词器-显示窗口.html`。

### 4. 双屏部署
将弹出的“显示窗口”拖拽到你的第二块屏幕（即朝向演讲者的提词器屏幕），并最大化/全屏显示。

### 5. 开始提词
在控制面板中新建或选择已有脚本，点击播放，即可开始提词！控制面板会实时显示当前的滚动进度和画面预览。

## 📂 文件结构

- `双屏提词器-控制面板.html`：提词器的主控端，供导播或操作人员使用，包含脚本管理、播放控制和画面预览功能。
- `双屏提词器-显示窗口.html`：提词器的显示端，供演讲者或主持人观看，提供纯净无干扰的黑底白字滚动文本界面。
- `README.md`：项目说明文档。
- `.gitignore`：Git 忽略文件配置。

## 🛠️ 技术实现

- **HTML5** / **CSS3**：用于构建界面和苹果风的扁平化样式。
- **Vanilla JavaScript (原生 JS)**：
  - 利用 `localStorage` 事件监听 或 `Window.postMessage` 实现跨标签页/窗口的实时双屏通信。
  - 使用 `requestAnimationFrame` 实现流畅的文本滚动效果。
